# Toronto Zoo Docs Layout

Use this directory for Toronto Zoo visit-specific artifacts and reusable local templates.

## Structure

- `templates/`: reusable prompts and reusable examples.
- `<YYYY-MM-DD>/`: per-visit artifacts.

Per-visit folders may include:

- `transcripts/`
- `media-manifest.json`
- draft post markdown
- title recommendation notes

## Naming Rules

- Use ISO date folders: `YYYY-MM-DD`.
- Keep transcript files ordered with numeric prefixes when sequence matters.
